Без консоли

Если установить небольшое Chromium-расширение из проекта, дальше на странице WB будет одна кнопка импорта без консоли и без ручной вставки JSON.

Папка расширения: browser-extension/wb-session-importer

  1. Нажмите кнопку выше.
  2. В новом окне войдите в WB под нужным номером.
  3. На странице WB откройте консоль браузера и вставьте скрипт.
  4. Скрипт скопирует JSON WB-сессии. Вернитесь в mini app и вставьте его в импорт.
