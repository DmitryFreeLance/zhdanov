Новый аккаунт WB

Номер нужен, чтобы бот нашёл и привязал нужную WB-сессию к этому чату.

Этот сценарий лучше делать с ноутбука или ПК: нужно открыть обычный браузер и вставить скрипт в консоль страницы WB.

Импорт готовой сессии

Войдите в wb.logistics в обычном браузере, скопируйте JSON сессии из консоли и вставьте его сюда. Бот сам привяжет JSON к номеру из поля выше.

Пошагово
Открыть WB Logistics
  1. Введите номер телефона аккаунта WB в поле выше.
  2. Откройте WB Logistics по кнопке выше на ноутбуке или ПК и войдите в нужный аккаунт.
  3. Нажмите Скопировать скрипт, откройте консоль браузера на странице WB и вставьте скрипт.
  4. Скрипт выведет или скопирует JSON сессии. Вернитесь в mini app и вставьте его в поле ниже.
  5. Нажмите Импортировать сессию. Бот сам подхватит JSON и привяжет его к номеру сверху.

Если скрипт вывел JSON в консоль, просто скопируйте его целиком и вставьте в это поле.
